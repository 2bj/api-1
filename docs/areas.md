# Регионы

### `GET /areas` — древовидный список всех регионов

```json
[
  {
    "name": "Россия",
    "id": "113",
    "parent_id": null,
    "areas": [
        {
          "name": "Дальневосточный округ"
          "id": "220",
          "parent_id": "113",
          "areas": []
        }
    ]
  }
]
```

### `GET /areas/{area_id}` — древовидный список регионов, начиная с указанного

```json
[
  {
    "name": "Центральный округ",
    "id": "232",
    "parent_id": "113",
    "areas": [
        {
          "name": "Москва"
          "id": "1",
          "parent_id": "232",
          "areas": []
        }
    ]
  }
]
```
